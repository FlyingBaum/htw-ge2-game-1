2D game group project using Unity 

## Gruppe gem-enjins:

### Main Mechanic:
* Blobby Volley like
* Lokale 2D 2 Spieler
* Steuerung für 2 Spieler
* Physikobjekte
* Physics Based
* Interaktion zwischen Spieler und Objekten
* 2 Plattformen
* Timer/Punkte als Gewinn/Verlier-Bedingung
* StartMenu und PausenMenu
* 1 level

### Should Have:

* Verschiedene Level

### NiceToHave:
* Character Typen
* Portale
* Animationen (Ästhetik)
* Bessere Physik
